### 조건문

- if 문

  - 단일 if

    ```
    if (조건식)
    	실행문장;
    ```

    ```
    if (조건식)
    	실행문장;
    else
    	실행문장;
    ```

    ```
    - 조건문에 문장을 두 개 이상 묶어서 매달고 싶으면 {} 블록을 이용해 명령문을 묶어준다.
    if (조건식) {
       ...
    } else {
       ...
    }
    ```

  - 다중if

    ```
    if (조건식) {
    	실행문장;
    } else if (조건식) {
    	실행문장;
    } else if (조건식) {
    	실행문장;
    } else {
    	실행문장;
    }
    ```

  - 내포된 if

    ```
    if (조건식) {
    	if (조건식)
    	 ...
    	if (조건식)
    	 ...
    	else
    	 ...
    }
    ```
