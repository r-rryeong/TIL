- itertools 모듈 사용
```python
from itertools import combinations

def check(a, b, c):
    total = a + b + c
    for i in range(2, total):
        if total % i == 0:
            return False
    return True

def solution(nums):
    answer = 0
    lst = list(combinations(nums, 3))
    for j in lst:
        if check(j[0], j[1], j[2]) == True:
            answer += 1
        
    return answer
```
